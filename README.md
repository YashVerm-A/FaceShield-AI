# Instant Privacy Filter (Real-Time Face Blurring)

## What the project does
This project is a real-time computer vision tool that automatically detects faces using a webcam and applies a strong blur filter over them. It acts as an "Instant Privacy Filter," which can be used to hide the identities of people in video feeds, protecting the privacy of bystanders in public recordings or live streams.

## How to set it up

You must have Python installed on your computer.

Clone or download this repository to your local machine.

Open your terminal or command prompt and install the required OpenCV library by running this command:
pip install opencv-python

## How to use it

Run the Python script from your terminal:
python face_blur.py

Your webcam will turn on automatically. Step in front of the camera, and your face will be blurred instantly.

To stop the program and close the camera, press the 'q' key on your keyboard.
